This is "Turkey Challenge" done in PHP vanilla
it adds, edit,delete,display,save to file, and load from file "Turkeys"


Requirements
PHP 7.4+ (or later)
MySQL 5.7+ (or later)

Ensure PHP & MySQL are installed

Run php -v to check PHP version.
Run mysql --version to check MySQL version.
Enable PDO (if not already enabled)


Modify Database Credentials If needed
Update the script (db_connect.php) with your database details:
$host = "localhost";
$dbname = "your_database_name";
$username = "your_username";
$password = "your_password";
